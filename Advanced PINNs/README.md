### This folder contains the solution to some physics problems using PINNs, the directory structure is like this:
#### `models` folder contains the trained keras models, `plots` folder contains the relevant plots like loss curves, PINN approximation and the `notebooks` folder contains the jupyter notebooks with the relevant impplementation.
#### Better code quality of the solution is shown in `projectile motion solution` and other solutions code quality is being improved.
#### The detailed setup, solution, ansatz and residual formulation explanation is given in the pdf document named `PINNS by Aditya Saroha`, please refer these as there are very less comments available in the notebooks.
